Rozpocząłeś/rozpoczęłaś już pracę nad dużym samodzielnym projektem. Chcesz, żeby twoja praca była bezpieczna i chcesz eksplorować różne podejścia do tematu.

Użyj repozytorium z poprzedniego ćwiczenia.

Jeśli jeszcze nie masz, stwórz konto na GitHubie: github.com/join

Póki co, twoje repozytorium istnieje tylko lokalnie. Stwórz nowe publiczne repozytorium: github.com/new, żeby spiąć je ze swoim lokalnym.

Przejdź na nowo utworzone repozytorium i skopiuj url HTTPS.

$ git remote add origin https://github.com/grazin/repo1.git

Wróć do git basha i zepnij swoje repozytorium z originem z GitHuba.

$ git push -u origin master

Teraz pusznij zmiany do origina (czyli do GitHuba). Ponieważ to pierwszy komit na tym branczu, użyj przełącznika -u. GitHub o ile już tego nie zrobiłeś/zrobiłaś poprosi cię o autentykację. Następnie znów odczytaj status. 

$ git log --graph --oneline
